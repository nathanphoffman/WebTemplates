ElePHPant Trampler in development

When I was originally considering how to release my open source code, I didn't want to have to deal with github or other projects, enter ElePHPant Packager. This packager toolset is designed to generate a single PHP file which contains an overview of the files that will be installed and the destinations they will be installed to. The PHP file contains all of the code of the package (meaning the packaged code must be ASCII non-encoded) So for example, TXT files, PHP files, HTML files, and JS files are all ok, but images like JPEGS are not.

ImageURIKit image uploads made easy

Dealing with image attachments can be frustrating, particularly if your user base is uploading large photos (say from a camera) over bad internet connections. We ran into this same issue where I work, and we simply didn't have the resources and time to spend figuring out a good solution. I decided I would take the time to develop it in my own time and release it as open source content. That way it would aid the community, not waste my time at work with a non-critical project, and ultimately benefit everyone in the end.

I can't take much credit for the idea and even about 30% of the code is from forums or other open sources.
